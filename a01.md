
## VSITE OOP zadatak 1

Napišite klase `rect` i `size` tako da sljedeći program: 

```C++ 
int main(){
    oop::rect r1(0, 0, 50, 50), r2(50, 50, 100, 100);
    r1 += oop::size(10, 10); // offset rect
    std::cout << (r1 & r2); // print intersection
}
```

ispisuje presjek dva pravokutnika:

`(50,50,60,60)`
